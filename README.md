### **üé¨ Multimedia & AI Tools**

* **AI Subtitle Translator:** Instantly translate .srt files into 20+ languages (Spanish, French, Chinese, Hindi, etc.) while preserving timecodes using Translation APIs.  
* **Voice to Text:** Real-time audio transcription using the Web Speech API.  
* **YouTube Thumbnail Downloader:** Extract HD (1280x720) thumbnails from any YouTube video URL.

### **üìÑ PDF & Document Tools**

* **Merge PDFs:** Combine multiple PDF files into a single document instantly.  
* **Images to PDF:** Convert a sequence of images into a PDF file.

### **üñºÔ∏è Image Tools**

* **JPG to PNG Converter:** High-speed local conversion without quality loss.

## **üß© Chrome Extension (Launcher)**

This project includes a "Launcher" Chrome Extension that provides quick access to the tools directly from your browser toolbar.

**Installation:**

1. Go to chrome://extensions/ in your browser.  
2. Enable **Developer Mode** (top right switch).  
3. Click **Load Unpacked**.  
4. Select the Extension folder from this repository.
